Arduino Integrated Hotel Management System

Designed and Developed by Yunus Emre Oflaz

Project Overview

This project was developed using Arduino and focuses on managing hotel operations such as check-in/check-out, room status tracking, and automated access control using RFID cards.

Features

  • RFID card-based room access
  • Room status display (occupied / available)
  • Guest registration and logging via LCD and buttons
  • Data tracking through serial monitor
